JSONLint
===============
This is a plugin for Sublime Text 2 that finds issues in JSON.

Installation
------------
1. `cd <packages directory>`
2. `git clone git@bitbucket.org:hmml/jsonlint.git`

Usage
-----
Annotations are enabled when syntax is set to JSON.
