# StripHTML
Super simple HTML stripping command. Press `Ctrl+Alt+H` over a selection to replace the text.

# Usage

## Key bindings
- **Windows & Linux:** CTRL+Alt+H
- **OSX** Super+Alt+H

Also available through the Command Palette.
