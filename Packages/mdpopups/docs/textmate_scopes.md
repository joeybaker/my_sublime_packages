## Supported Scopes

- comment.line.double-slash
- comment.line.double-dash
- comment.line.number-sign
- comment.line.percentage
- comment.line.character
- comment.block

- constant

- constant.language
- constant.other
- entity

- entity.name.type
- entity.name.tag

- entity.other

- invalid

- invalid.deprecated
- keyword

- keyword.operator
- keyword.other
- markup

- markup.bold
- markup.heading
- markup.italic
- markup.list

- markup.list.unnumbered
- markup.quote
- markup.raw
- markup.other

- string
